Предсказание по общему суффиксу
На этапе генерации бинарного представления строим автомат на парах типа <ReverseSuffix(X),A>, где ReverseSuffix(X) - перевернутый суффикс словоформы Х некоторой фиксированной длины, а А - это аннотация Х.
Идем с конца ненайденного слова по автомату до тех пор, пока существует состояние, в которое можно перейти, используя текущую букву слова. Далее обходом графа собираем все достижимые аннотации.
Факторизуем полученные аннотации по частям речи и выдаем результаты только по продуктивным  частям речи.
