Размер автомата: необходима минимизация
 Минимизация автомата – построение меньшего автомата, из которого выводимо  то же множество строк.
  Неминимизированный  автомат занимает много места, например, для русского языка неминимизированный  автомат - это 70 млн. связей, а минимизированный - 1 млн. связей.
